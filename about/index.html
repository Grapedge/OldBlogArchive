作品

以下是我个人作品的展示（MOD等没有计入其中）：

落月之紫水晶传奇

讲述了落月星人冷夜新魂的故事。原本落月星在冷夜新魂的统治下，变的十分繁荣昌盛，可是某一天落月星球之宝紫水晶突然神秘消失，紫水晶关系着落月星未来，无奈之下冷夜新魂开始了寻找紫水晶的冒险之旅。

这是我的第一部 RM 游戏，当初我以落月论坛版主的身份制作了此游戏，但因为当初技术等方面的不足，这个游戏也成了我的黑历史（问题是某些奇奇怪怪的网站把这个游戏收录了，我有什么办法，哭）。

S-Maker游戏引擎

S-Maker 是用于制作 战棋 游戏的工具，其实现基本仿照 三国志曹操传，曾在某内部群内流传了一段时间，后因为一些原因停止更新。

编辑器是使用 C# 编写的，引擎是使用 C++ + DX9 编写的，永远记得当初渲染一个窗体花了一天的时间。

小石子大冒险

描述的是小石子闯关的故事。


这个是当初帮助朋友做互联网服务时的作品，因为站长的昵称叫小石子，这部作品也算是黑历史了。

天真的冒险3D版

一部 ARPG 类型的游戏，讲述的是主角 天真 寻找丢失的女主（忘了叫什么了），结果女主离他远去最后死去的故事。

这个游戏做的我当初都想给自己寄刀片了…

逝去之城

昔日美好的城镇遭到了破坏，你需要完成一系列近乎不可能完成的任务。


逝去之城 是我早期作品中做的最用心的一部，因为它和我当时在网络上的 ID ，我个人游戏论坛的名字是相同的。非常可惜的是这部作品因为电脑损坏等只留下来了这两张我传到 QQ空间 里的图片（当时还不会用 github）。

天真的冒险2D版

剧情和3D版基本一样。


这部作品是中考结束后用于参加当时 66RPG 的短十的作品，由于已经有了大量的游戏开发经验，这部作品开发耗费了不到一周的时间。这部游戏的封面是我从3D版中直接截的一个画面，所以被很多人吐槽画风了。

天真的冒险TPS版

剧情和2D版基本一样。

就是把3D版的 ARPG 的操作改成了 TPS，其他基本没有什么区别。

UltramanX

一个横版战斗的艾克斯奥特曼打怪兽的小游戏。

是我使用 Unity 制作 2D 游戏的第一个尝试，当时的 Unity 制作 2D 游戏远没有今天这么方便，比如当时使用 Cube 来实现精灵的渲染而不是今天的 Sprite。
这部作品也是当时新出的艾克斯奥特曼的个人爱好型作品。

班级纪念册


高中毕业做的班级纪念册，上图是用美图秀秀做的一个产品说明书。
